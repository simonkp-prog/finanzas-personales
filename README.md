# Finanzas personales

App para registrar gastos desde el teléfono, basada en el sistema de seguimiento que mantengo en Excel.

## ¿Qué hace?

- Registrar un gasto en menos de 30 segundos desde el celular
- Categorías: Gastos del mes, Deseos, Ahorro
- Resumen mensual con presupuesto vs real por categoría
- Compatible con el Excel existente

## Estructura del proyecto

- `Registro finanzas personales.xlsx` — Excel original con historial de registros e inversiones
- `app/` — App web progresiva (PWA) para el teléfono (en desarrollo)

## Roadmap

- [ ] PWA con registro de gastos en < 30 segundos
